# Raw Idea

**Date**: 2026-01-10

## Feature Description

Redesign how the review page looks with the CAWPILE ratings. The user wants to have each letter larger and in a box with the letter word under it, and bring the rating next to this box.

## Original Request

The user wants to redesign the review page's CAWPILE rating display:
- Each letter should be larger
- Each letter should be in a box
- The full word for each letter should appear under the letter
- The rating value should be positioned next to the box

## Context

This relates to the Cawpile rating system which has 7 customizable facets:
- Fiction: Characters, Atmosphere, Writing, Plot, Intrigue, Logic, Enjoyment
- Non-Fiction: Credibility, Authenticity, Writing, Personal Impact, Intrigue, Logic, Enjoyment

Each facet has a letter abbreviation (C, A, W, P/P, I, L, E) and the redesign aims to make this more visually prominent and readable.
