# Raw Idea: React Native Mobile App

## Original Description

"Set up a React Native mobile version of the webapp"

## Context

This feature is for the Cawpile project — a book reading tracker with a custom CAWPILE rating system.

The existing web application is built with:
- Next.js 16 (App Router), React 19, TypeScript, TailwindCSS 4
- Prisma ORM with PostgreSQL (Neon serverless)
- NextAuth v5 with Google OAuth
- Multi-provider book search (Google Books, Hardcover, IBDB, local DB)
- Reading progress tracking, CAWPILE ratings, analytics charts
- Monthly recap video generation (Remotion + Express service)
- S3 for avatar/cover image storage
